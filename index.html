Tiny Crossing

Welcome to Tiny Crossing, a game developed by Andrei Borissenko as part of the Udacity Front-End Web Developer Nanodegree.

Instructions

  • Use cursor keys to choose an icon and press enter to select it. Then use cursor keys to move the icon around the map.
  • Avoid the enemies.
  • Avoid the water by stepping on corn. NOTE: if the corn seems to be bunching up and is difficult to cross, refresh the browser, it should fix the problem.
  • Collect three keys to advance to a new, harder round.
  • Beat the time for additional points.
